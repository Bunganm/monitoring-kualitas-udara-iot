# 🌫️ Implementasi Sistem Monitoring dan Visualisasi Kualitas Udara pada Ruang Kerja Pribadi Berbasis Internet of Things

## 📌 Deskripsi Proyek

Proyek ini merupakan implementasi sistem monitoring kualitas udara berbasis Internet of Things (IoT) yang dikembangkan menggunakan ESP32, sensor DHT22, dan sensor MQ2. Sistem dirancang untuk memantau kondisi lingkungan ruang kerja pribadi melalui pengukuran suhu, kelembapan, dan kualitas udara.

Data hasil monitoring divisualisasikan menggunakan dashboard berbasis Streamlit yang menyediakan tampilan interaktif berupa indikator monitoring, grafik visualisasi, heatmap korelasi, dan analisis dataset Air Quality UCI.

---

## 🎯 Tujuan

- Memantau kondisi lingkungan ruang kerja secara real-time.
- Menampilkan data kualitas udara dalam bentuk dashboard interaktif.
- Memanfaatkan teknologi Internet of Things (IoT) untuk monitoring lingkungan.
- Melakukan visualisasi dan analisis data kualitas udara.
